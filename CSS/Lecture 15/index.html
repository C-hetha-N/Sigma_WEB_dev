There are three ways of adding CSS

  1. Inline CSS
  2. Internal CSS
  3. External CSS
  • Inline and Internal css are written in the .html file
  • Inline - for each line using STYLE ATTRIBUTE
  • Internal - for whole html file of same tag line using STYLE TAG
  • External - Internal content is written in the .css file and linked with html
  • Inline - small projects but not recommended
  • Internal - small projects or medium projects with less css code while using bootstrap, tailwind and css frameworks
  • External - Mostly recommended, specifically for Big projects
